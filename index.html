Naviola — Responsive Multi-level Navigation

Naviola is a navigation menu, has multi-level supports, where parent item can be clickable (instead of hashed). Dropdowns can be opened using toggle buttons in smaller devices, while in larger devices it be opened by just hovering over the parent menu.

Features

  • Unlimited Levels – we suggest using maximum 3-level deep for better UX
  • Parent menu is clickable – instad of putting hash (#) you can put link
  • Submenu opens on toggle button click in smaller viewport
  • Submenu opens on hover in wider viewport
  • Minimal code-setup – no extra class, no markup - only on the parent one
  • JavaScripts only – no jQuery required
  • Accessibile! – close, but not yet üôÅ

Setup

01. Stylesheet in <head></head>

<head>
    <link rel="stylesheet" href="navigation.min.css">
</head>

02. JavaScript in Footer

    <script src="navigation.min.js" type="text/javascript"></script>
</body>

03. Navigation Markup

<nav class="navigation navigation-medium">
    <ul>
        <li><a href="#">Menu item</a></li>
        <li>
            <a href="#">Menu item</a>
            <ul>
                <li><a href="#">Submenu item</a></li>
            </ul>
        </li>
    </ul>
</nav>

How to Use

Side Navigation

By default the navigation is a mini one, applicable for sidebar menu. So, only a .navigation class is enough for the menu.

<nav class="navigation"></nav>
